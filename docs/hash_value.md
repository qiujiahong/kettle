# 唯一行(哈希值)

## 案例介绍

唯一行hash值，就是删除数据流中重复的行。  
和排序+去除重复记录类似，但实现原理不同，该方法效率会高一些。

从excel中读取数据，去除重复的字段，并保存到excel。
