# switch case 

## 案列说明

![](./assets/2019-06-09-23-30-35.png) 


从excel中读取数据，按SEX进行分离数据分类，把女性、男性和保密保存在不同的EXCEL文件中  
* 1表示男性
* 0表示女性
* 2表示保密
